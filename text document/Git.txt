git بخش های مهم گیت برای یاد گیری 
1 = git --version یا git -v برای نشان دادن ورژن نصب شده گیت بر روی سیستم است.
2 = git --help یا git -h برای نشان دادن تمامی ابزارت برای استفاده از گیت و گیت هاب.
3 = git init برای این استفاده می شود گه باید مخزنی برای پروژه بوجود بیاوریم.
4 = git status برای نشن دادن فایل هایی که هنوز add نشده اند را نشان میدهد تا آن ها را add کنیم.
5 = git add برای add کردن تمامی فایل های پروژه .
add هم به صورت کامل انجام پذیر است هم به صورت تک تک که کد آن به این شکل است 
git add. یا git index.html این یک متقیر است 
6 = git commit -m "مسییج مورد نظر"  
6 = برای ثبت تمامی فایل ها بر روی گیت.
7 = git commit -a -m تمامی کار ها از جمله add و commit کردن را برای ما انجام میدهد.
8 = git rm --cached -r . برای برگشت دادن فایل ها از قسمت add شده ها.
9 = git log برای نشان دادن commit ها هستش.
10= git diff برای نشان دادن اینکه چه اتفاقاتی بر روی پروژه روخ داده است .
11= git log --oneline برای خطی نشان دادن commit ها استفاده میشود .
12= git log -تعداد commit  --oneline برای تعداد زیاد commit استفاده میشود.
13= git log -p برای نشان دادن اینکه چه اتفاقاتی بروی پروژه روخ داده هم زمان با commit ها نشان داده میشود . 
13 = برای بیرون آمدن از git log -p باید دستور q را بزنید.
14= git reset کد commit ها .
15= git restore -- . 
16= git checkout -- . 
17= git stash save "massege" / git stash / git stash list / git stash drop / git stash show / git stash apply / git stash pop /  
18= git rm --cached -r . برای برگشتوندن فایل هایی که تو ریپازوتوری هستند انجام میگیرد
19= touch .gitignore برای به وجود آوردن gitignore انجام میگیرد
